# Blackjack Project

Blackjack 游戏计算器和模拟器项目。

## 项目结构

- `crates/calculator` - Blackjack 计算器核心库
- `apps/simulator` - Blackjack 模拟器应用

## 开发

```bash
# 安装依赖
pnpm install

# 构建项目
pnpm build

# 运行开发环境
pnpm dev

# 格式化代码
pnpm format

# 运行测试
pnpm test
```

## Rust 开发

```bash
# 构建 Rust 项目
cargo build

# 运行测试
cargo test

# 运行 simulator
cargo run --bin simulator
```
